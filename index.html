This is an evolution simulator that uses neural networks controlled by a genetic algorithm to evolve blue creatures whose goal it is to learn how to eat the green food.

There are a variety of simulation parameters that you can change:

  1. Number of creatures: this is the total numbers of creatures in the simulation.
  2. Number of food: this is the total number of food in the simulation.
  3. Detection radius: this is the radius in which the creatures can see food.
  4. Iterations: this is how long eat evaluation/generation will last for.
  5. Mutation rate: the percentage of creatures that will be mutated every generation. Note, This is a value between 0 and 1 e.g. 0.5 = 50%.
  6. Iterations: this is how long each evaluation/generation will last for.
  7. Number of hidden nodes at start: how many hidden nodes the neural networks will have when initialised.
  8. Elitism percentage: the percentage of the top performing creatures that get copied into the next generation. This is also a decimal from 0 to 1.
  9. Mutation amount: the amount of times a mutation method will get applied to each neural network.
